What are SRT and VTT used for in video projects?

\n\n

SubRip (SRT) is a plain text format popular for subtitles. It uses timestamps like 00:01:15,500 and simple cue blocks. WebVTT (VTT) is a modern standard for HTML5 video captions. It starts with a WEBVTT header and uses dots in time codes like 00:01:15.500.

\n\n

SRT is common in TV and local players. VTT fits web pages, streaming services, and online players that support native caption tags in HTML5.

\n\n

How can you convert srt to vtt with FFmpeg?

\n\n

FFmpeg offers a free, cross platform command line tool. To convert your file, open a terminal or command prompt

\n\n
    \n
  • Command: ffmpeg -i input.srt output.vtt
    • \n
\n\n

This single command reads your SRT file and writes a VTT file with a valid header. FFmpeg automatically adds the WEBVTT header and adjusts timestamp format.

What are SRT and VTT subtitle formats?

SRT is a plain text subtitle format using sequential cues and timecodes in Hours:Minutes:Seconds,Milliseconds separated by commas. It works with many legacy players and simple editors.

VTT (WebVTT) adds a mandatory WEBVTT header and uses dots in timecodes. It integrates natively with HTML5 video, supports basic CSS styling, positioning, and accessibility tags.

How to convert SRT to VTT using FFmpeg commands?

FFmpeg is a versatile tool for converting media formats. To convert a single file, use:

  • ffmpeg -i input.srt output.vtt

To batch convert on Mac or Linux, run:

  • for f in *.srt; do ffmpeg -i "$f" "${f%.srt}.vtt"; done

How to convert SRT to VTT using a Python script?

You can add a WEBVTT header and replace commas with dots in timecodes. Example:

  • with open('input.srt') as f: lines = f.read()
  • with open('output.vtt','w') as out: out.write('WEBVTT\n\n' + lines.replace(',', '.'))

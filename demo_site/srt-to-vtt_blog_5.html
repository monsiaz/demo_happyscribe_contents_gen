How to add your SRT file to an Adobe Premiere project

Importing the SRT file

First, open your Premiere project. If you don’t have an SRT yet, get one via our Automatic transcription or order Human-made subtitles for top accuracy.

  1. Choose File > Import and select your .srt file.
  2. Drop the SRT onto your project panel. Premiere recognizes it as a captions file.
  3. Drag the SRT clip onto your timeline above your video track.

Now you have captions in place. You can preview and adjust them in the Caption panel. If it doesn’t appear, go to Window > Text > Captions.

Checking the captions panel

  • Open the Captions workspace to see text and timing.
  • Play the sequence to confirm captions align with speech.
  • Use the Subtitle editor in Premiere to tweak any line breaks or overlaps.

At this point, your SRT is integrated. Next, you’ll export it as a VTT file for your web team or HTML5 player.

How to export your captions as a VTT file

Selecting the sequence

With your captions in place, select the sequence you want to export. Make sure your timeline is active by clicking anywhere on it. Premiere needs to know which sequence holds the captions you’ll convert.

Export settings

  1. Go to File > Export > Media.
  2. In the Export Settings dialog, switch Format to “Subtitles”.
  3. Under Caption settings, set Caption format to “WebVTT”.
  4. Choose “Create Sidecar File” if you want the VTT separate from your video.
  5. Click Export to generate your .vtt file.

Premiere writes a standard VTT header, timecodes, and cues. You now have a file named like sequenceName.vtt ready for web players.

Tips for smooth timing and style

Exporting VTT is only half the battle. Here are best practices to avoid common mistakes:

  • Check timecodes for any overlap or zero-length captions.
  • Remove extra spaces at the start or end of lines.
  • Verify the VTT header reads “WEBVTT” on the first line.
  • Break long sentences into shorter cues for readability.
  • Test in your browser by dragging the VTT onto an HTML5 video.

If you spot issues, you can edit your VTT in any text editor or go back to Premiere’s Caption panel. For bulk edits, consider a quick script or use our API documentation to automate fixes.

How to prepare your SRT file for smooth conversion

Before you start converting check that your SRT file meets these basic requirements:
  • Plain text format encoded in UTF-8 or ANSI.
  • Numeric cue numbers starting at 1 and incrementing by one.
  • Timestamps in hours:minutes:seconds,milliseconds style.
  • No extra blank lines beyond the separation between captions.
If your file comes from a transcription service such as Automatic transcription you may already have a clean SRT. If you ordered Human-made subtitles double check encoding and formatting. A few minutes of prep will save time when editing the VTT file.

Step by step guide to convert srt to vtt in Notepad

It is easy to convert SRT to VTT using the built‐in Notepad editor. Follow these steps:
  1. Right click your SRT file and choose Open with Notepad.
  2. At the very top of the document type: WEBVTT and press Enter twice.
  3. Find all comma characters in your timestamps and replace them with a period. In Notepad use Edit > Replace to change , into ..
  4. Save the file with a .vtt extension. Choose File > Save as, set Save as type to All files, then name it yourfile.vtt.
Now you have a valid VTT file that any browser or video player can read. You did not need to install other software.

Common mistakes when editing VTT files by hand

Even a simple manual conversion can run into pitfalls. Watch out for these issues:
  • Missing header. Every VTT needs WEBVTT on the first line.
  • Wrong timestamp format. VTT uses 00:01:23.456 not 00:01:23,456.
  • Encoding errors. Save as UTF-8 if you have special characters or accents.
  • Extra spaces before timestamps. They can break parsing in some players.
  • No blank line after the header. Most players require a clear line to start cues.
Spotting these early stops playback errors and caption glitches. A quick preview in your browser will flag most problems.

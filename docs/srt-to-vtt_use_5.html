how do you automate conversion with the HappyScribe API?

For teams managing dozens of videos each month, a manual process can become a bottleneck. Here’s how you can set up a script to convert SRT to VTT automatically:

  1. Retrieve your API key from the HappyScribe dashboard under “API access.”
  2. Install an HTTP client library in your favorite language (for example, Node.js or Python).
  3. Write a script that uploads your SRT file, requests a VTT conversion, and downloads the result.

Here’s a basic Node.js example:

const axios = require('axios');
const fs = require('fs');

async function convertSrtToVtt(apiKey, srtFilePath) {
  // 1. Upload the SRT file
  const uploadRes = await axios.post(
    'https://api.happyscribe.com/v1/uploads',
    fs.createReadStream(srtFilePath),
    { headers: { 'Authorization': `Bearer ${apiKey}`, 'Content-Type': 'application/octet-stream' } }
  );

  const fileId = uploadRes.data.id;

  // 2. Request conversion
  const convertRes = await axios.post(
    'https://api.happyscribe.com/v1/subtitles',
    { audio_file_id: fileId, format: 'vtt' },
    { headers: { 'Authorization': `Bearer ${apiKey}` } }
  );

  const subtitleId = convertRes.data.id;

  // 3. Download the VTT file once ready
  const downloadRes = await axios.get(
    `https://api.happyscribe.com/v1/subtitles/${subtitleId}/download`,
    { headers: { 'Authorization': `Bearer ${apiKey}` }, responseType: 'stream' }
  );
  
  const outputPath = srtFilePath.replace('.srt', '.vtt');
  downloadRes.data.pipe(fs.createWriteStream(outputPath));
  console.log(`Converted file saved to ${outputPath}`);
}

// usage
const apiKey = 'YOUR_API_KEY';
convertSrtToVtt(apiKey, 'training-session.srt');

This script handles the upload, conversion, and download steps. You can integrate it into your video publishing pipeline so that every new training asset gets its VTT file automatically.

Film festival subtitle prep: SRT to VTT conversion

When you’re gearing up for a film festival premiere, every detail matters. You’ve got your stunning visuals, your compelling sound design, and your passionate storytelling. But there’s one piece you can’t afford to get wrong: subtitles. A glitchy subtitle file can distract the audience, throw off the pacing, or worse, lead to misinterpretation of key dialogue. If your festival requires a WebVTT (.vtt) file but your team is working off SubRip Text (.srt) files, you need a rock-solid, repeatable process to convert between formats without missing time codes or messing up special characters.

In this guide you’ll learn exactly how to streamline SRT to VTT conversions using HappyScribe. We’ll walk through a realistic scenario, show you a step-by-step workflow, highlight three HappyScribe features that will save you time, and answer common questions in a Mini FAQ. By the end, you’ll have a clear plan to handle subtitle prep like a pro—and deliver perfectly synced VTT files for your next festival submission.

Why convert SRT to VTT for film festivals?

  • Compatibility: Many online screening platforms and digital cinema packages require VTT for captions and metadata.
  • Formatting: VTT supports styling, positioning, and Web standards that keep your subtitles looking crisp on any device.
  • Automation: Once your pipeline is set up, batch conversions of dozens of films become fast and error free.

How does the workflow look with HappyScribe?

Here’s a step-by-step guide to convert your SRT files into ready-to-go VTT files, all while keeping track of version control and quality checks. Follow along and adapt each step to your team’s needs.

Step 1: gather your source files

  • Create a dedicated folder called festival_subtitles.
  • Place each film’s .srt subtitle file inside a subfolder named after the project (for example, MyShortFilm).
  • Keep a simple text file metadata.csv with columns: Project, Language, Submit_as.

Step 2: install HappyScribe API client

You’ll need Node.js or Python. Here’s how to set up the Node.js client:

# in terminal
mkdir subtitle-workflow
cd subtitle-workflow
npm init -y
npm install happyscribe-api

Step 3: authenticate and upload SRT

Create a file upload.js and add:

const HappyScribe = require('happyscribe-api');
const fs = require('fs');

const client = new HappyScribe({
  apiKey: process.env.HAPPYSCRIBE_API_KEY
});

async function uploadSubtitle(projectName, filePath) {
  const response = await client.subtitles.upload({
    project: projectName,
    source_file: fs.createReadStream(filePath),
    format: 'srt'
  });
  console.log('Uploaded:', response);
  return response.id;
}

// example usage
uploadSubtitle('MyShortFilm', './festival_subtitles/MyShortFilm/en.srt');

This script uploads an SRT file into HappyScribe’s system where you can track its status.

Step 4: trigger the format conversion

Once uploaded, call the conversion endpoint:

async function convertToVtt(subtitleId) {
  const job = await client.subtitles.convert({
    subtitle_id: subtitleId,
    target_format: 'vtt'
  });
  console.log('Conversion job started:', job.id);
  return job.id;
}

// chain upload and convert
uploadSubtitle('MyShortFilm', './festival_subtitles/MyShortFilm/en.srt')
  .then(convertToVtt)
  .catch(console.error);

Step 5: poll for completion and download

Create a simple poller poll.js:

async function pollJob(jobId) {
  while (true) {
    const status = await client.jobs.get(jobId);
    console.log('Job status:', status.state);
    if (status.state === 'completed') {
      const download = await client.jobs.download({
        job_id: jobId,
        format: 'vtt'
      });
      console.log('Download link:', download.url);
      break;
    }
    await new Promise(r => setTimeout(r, 5000));
  }
}

// example usage
pollJob('your_job_id_here');

When the state changes to completed, you’ll get a secure URL to fetch the VTT file. You can script an automated download and move the file into your project folder, naming it MyShortFilm-en.vtt.

Step 6: review in the subtitle editor

  • Log into your HappyScribe account and open the subtitle editor.
  • Check for timing shifts, special characters, or punctuation errors.
  • Use the playhead to preview subtitles against your film reel.
  • Make edits right there; HappyScribe tracks every change.

Understanding SRT and VTT formats

What is SRT?

The SubRip subtitle format (SRT) is a plain text file that contains sequential subtitle entries. Each entry has an index, a timecode in HH:MM:SS,MS format, and the subtitle text. SRT is widely supported by video players, editing software such as Premiere, and online platforms.

What is VTT?

Web Video Text Tracks (VTT) uses the .vtt extension and supports features like styling, positioning, and metadata. VTT is required for HTML5 elements and is compatible with streaming services and browser-based caption generators.

Methods to convert srt to vtt

You can convert srt to vtt using online tools, command-line utilities, or manual editing.

Use the HappyScribe API for online conversion

You can call the HappyScribe API endpoint to upload an SRT file and receive a VTT output. This free convert srt to vtt endpoint requires an API key. Example using curl:

  • curl -X POST -H "Authorization: Bearer YOUR_API_KEY" -F "file=@subtitles.srt" https://api.happyscribe.com/v1/convert/srt-to-vtt -o subtitles.vtt

Manual conversion using a text editor

You can rename the file extension from .srt to .vtt and add the file header. Open in Notepad or any text editor and prepend "WEBVTT" on the first line, then save as .vtt. This method is free and works offline.

Command-line conversion with FFMPEG

Use ffmpeg in the terminal: ffmpeg -i subtitles.srt subtitles.vtt. This single command batch converts files and integrates with scripting. FFMPEG must be installed and in your PATH.

Convert srt to vtt with a Python snippet

Load the SRT file, replace commas in timecodes, add the header. Example Python script:

  • import re
  • with open('subtitles.srt', 'r') as f: data = f.read()
  • vtt = 'WEBVTT\n\n' + re.sub(r'(\d{2}),(\d{3})', r'\1.\2', data)
  • with open('subtitles.vtt', 'w') as f: f.write(vtt)

Ensuring compliant VTT subtitle playback

Validate timing and format

After conversion, use the Subtitle editor on HappyScribe or online validators to check timecode accuracy and styling tags. VTT requires dot notation in timecodes and optional cues for positions.

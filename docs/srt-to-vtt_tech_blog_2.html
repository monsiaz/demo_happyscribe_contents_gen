Automating SRT → VTT subtitle conversion in CI/CD

Manual conversion of subtitle files can introduce errors and slow down release cycles for video platforms. Lack of a repeatable process adds risk and increases support overhead. Integrating a reliable conversion step is critical to maintain quality and speed for global content workflows.

This article shows a practical approach to build an end to end automation for SRT to VTT conversion in a continuous integration and delivery pipeline. We will use the HappyScribe API to handle uploads, exports and downloads at scale. Senior developers and DevOps engineers will learn how to embed these steps in a CI/CD pipeline with code examples.

import os
import requests

api_key = os.environ.get("HAPPYSCRIBE_API_KEY")
headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = requests.get(
    "https://www.happyscribe.com/api/v1/transcriptions",
    headers=headers
)

print(response.json())

How to upload an SRT file and request a VTT export?

This example uploads an SRT file and requests a VTT output.

# 1) Obtain a signed URL for upload
curl -s -H "Authorization: Bearer $API_KEY" \
  "https://www.happyscribe.com/api/v1/uploads/new?filename=subtitle.srt" \
  | jq -r .signedUrl > upload_url.txt

# 2) Upload the SRT file
curl -s -X PUT -T subtitle.srt "$(cat upload_url.txt)"

# 3) Request VTT export
curl -s -X POST "https://www.happyscribe.com/api/v1/exports" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "export": {
    "format": "vtt",
    "transcription_ids": ["${TRANSCRIPT_ID}"]
  }
}'

How to integrate conversion into a CI/CD pipeline?

convert_subtitles:
  stage: conversion
  image: curlimages/curl:latest
  variables:
    HAPPYSCRIBE_API_KEY: "${HAPPYSCRIBE_API_KEY}"
  script:
    - echo "Upload SRT and convert to VTT"
    - SIGNED_URL=$(curl -s -H "Authorization: Bearer $HAPPYSCRIBE_API_KEY" \
        "https://www.happyscribe.com/api/v1/uploads/new?filename=video.srt" \
        | jq -r .signedUrl)
    - curl -s -X PUT -T video.srt "$SIGNED_URL"
    - EXPORT_ID=$(curl -s -X POST \
        -H "Authorization: Bearer $HAPPYSCRIBE_API_KEY" \
        -H "Content-Type: application/json" \
        -d '{"export":{"format":"vtt","transcription_ids":["'$TRANSCRIPT_ID'"]}}' \
        https://www.happyscribe.com/api/v1/exports \
        | jq -r .id)
    - |
      until [ "$(curl -s -H "Authorization: Bearer $HAPPYSCRIBE_API_KEY" \
         https://www.happyscribe.com/api/v1/exports/$EXPORT_ID \
         | jq -r .state)" = "ready" ]; do
        sleep 5
      done
    - curl -L "$(curl -s -H "Authorization: Bearer $HAPPYSCRIBE_API_KEY" \
        https://www.happyscribe.com/api/v1/exports/$EXPORT_ID \
        | jq -r .download_link)" -o video.vtt
  only:
    - main

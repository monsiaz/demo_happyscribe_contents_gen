Why convert SRT to VTT for your streaming platform?

If your team is delivering subtitles to a global streaming service, chances are the platform requires WebVTT files (VTT). While SRT files are universal and easy to work with, VTT adds more detail for timing and styling that modern players rely on:

  • Precise cue settings for background color, positioning, and alignment.
  • Support for speaker names, descriptions, and styling tags.
  • Compatibility with HTML5 players and streaming protocols like HLS and DASH.

Without VTT, captions may appear off-screen, display at the wrong time, or lack the style rules your brand demands. By converting to VTT, you’ll ensure a consistent viewing experience and meet the spec your platform requires.

How does this help your team?

  • You’ll reduce manual editing time by up to 70 percent.
  • You’ll avoid formatting errors that delay uploads and reviews.
  • You’ll give your QA team a predictable file standard to check against.

What scripts can automate this?

Here’s a simplified Node.js example that uploads an SRT and converts it to VTT:

const fs = require('fs');
const fetch = require('node-fetch');
const apiKey = 'YOUR_API_KEY';

async function convertSrtToVtt(filePath) {
  const file = fs.createReadStream(filePath);
  // Step 1: upload file
  let response = await fetch('https://api.happyscribe.com/v1/upload', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${apiKey}` },
    body: new FormData().append('file', file)
  });
  const { job_id } = await response.json();

  // Step 2: convert to VTT
  response = await fetch('https://api.happyscribe.com/v1/convert', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ job_id, format: 'vtt' })
  });
  const { download_url } = await response.json();

  // Step 3: download file
  const downloadRes = await fetch(download_url);
  const vtt = await downloadRes.text();
  fs.writeFileSync(filePath.replace('.srt', '.vtt'), vtt);
  console.log('Converted', filePath);
}

convertSrtToVtt('ShowName_S01E01_en.srt');

Post-production batch captioning workflow

In any media production environment, teams often need to convert large volumes of subtitle files from one format to another without manual intervention. Whether you are working on a series of marketing videos, e-learning modules or social media clips, a reliable batch process saves time and ensures consistency.

This workflow focuses on converting SRT subtitle files into VTT format at scale. By leveraging the HappyScribe Product API, engineers can automate the entire pipeline, from secure uploads to parallel exports. The result is a streamlined post-production step that integrates easily into CI/CD pipelines or media asset management systems.

for ID in $ID1 $ID2; do
  while true; do
    STATE=$(curl -s -H "Authorization: Bearer $API_KEY" \
      https://www.happyscribe.com/api/v1/transcriptions/$ID \
      | jq -r .state)
    if [ "$STATE" = "automatic_done" ]; then
      echo "Transcription $ID ready"; break

    sleep 5
  done
done
